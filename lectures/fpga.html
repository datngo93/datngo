Syllabus

Topics

The course will focus on the following categories:

  1. Introduction to digital system design
    • Integrated circuit fabrication
    • Device technologies
    • Comparison of technologies
  2. Overview of FPGA and EDA software
    • FPGA
    • EDA software
    • Development flow
    • Tutorials
  3. Combinational circuit design
    • Operators
    • Combinational circuits
    • Code development
  4. Sequential circuit design
    • Basic memory elements
    • Synchronous circuits
    • Timing analysis
    • Code development
  5. Finite state machine
    • FSM representation
    • Mealy and Moore machines
    • FSM code development
  6. Selected topics
    • RAM
    • FIFO buffer
